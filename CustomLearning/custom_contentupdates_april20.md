# <a name="microsoft-365-learning-pathways-content-updates-for-april"></a>Microsoft 365 Atualizações de conteúdo de caminhos de aprendizado para abril
Os percursos de aprendizagem se esforçam para aderir a um ciclo de atualização de conteúdo trimestral. A lista a seguir das atualizações de conteúdo para abril de 2020.

## <a name="april-2020-content-updates"></a>Atualizações de conteúdo de abril de 2020
 Em resposta aos eventos atuais, a Atualização de Conteúdo de abril de 2020 fornece playlists de aprendizado e recursos projetados para habilitar o trabalho remoto em sua organização. Adicionamos um novo conjunto de playlists de Trabalho remotamente, atualizamos significativamente o conteúdo Microsoft Teams, especialmente para reuniões Teams, adicionamos um novo conjunto de Office para as playlists da Web e adicionamos novos ativos do Microsoft Forms. Como parte da atualização, também fornecemos uma lista de recursos de Trabalho Remoto para administradores.  
 
 Para saber mais sobre como as atualizações de conteúdo são aplicadas e como gerenciar atualizações de conteúdo, consulte [Manage Content Updates](custom_contentupdatesmanage.md). A seguir, lista as atualizações de conteúdo de Microsoft 365 de Microsoft 365 de abril de 2020. 

### <a name="office-365-was-rebranded-to-microsoft-365"></a>Office 365 foi renomeado para Microsoft 365
Office 365 foi renomeado para Microsoft 365. A alteração de nome é refletida em todo o Microsoft 365 conteúdo de caminhos de aprendizagem.
